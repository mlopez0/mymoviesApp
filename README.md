# mymoviesApp
Small Movies App developed with Flutter and developed within 48 hours



## Requirements ##

- Open the config.dart file in lib folder
- Add your TMDB api key

```
String getApiKey() {
  return 'YOUR_TMDB_API_KEY';
}
```


## How to run the application? ##

  

```
flutter run
```
